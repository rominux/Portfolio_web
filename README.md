# Portfolio Romain Lefebvre

Portfolio universitaire étudiants en BUT Informatique à l'IUT de Lille.

## Git

Ce dépôt Git contient le code source du portfolio déployé sur GitLab Pages.

## Déploiement

Le site est automatiquement déployé sur GitLab Pages via GitLab CI à chaque push sur la branche `master`.

**Lien du portfolio:** https://romainlefebvre.pages.gitlab.io

## Structure

- `public/` - Fichiers servis par GitLab Pages
  - `index.html` - Page d'accueil
  - `style.css` - Styles
  - `Projet/S1/` - Projets Semestre 1
  - `Projet/S2/` - Projets Semestre 2
  - `Projet/autre/` - Projets personnels
